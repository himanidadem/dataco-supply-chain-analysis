# Data Directory

## Dataset: DataCo Smart Supply Chain

The raw dataset is **not included in this repository** due to file size (~91 MB) and redistribution policy.

### How to Get the Data

1. Visit: [https://data.mendeley.com/datasets/8gx2fvg2k6/5](https://data.mendeley.com/datasets/8gx2fvg2k6/5)
2. Download the full dataset (zip file)
3. Extract and place these two files in `data/raw/`:
   - `DataCoSupplyChainDataset.csv` — main dataset
   - `DescriptionDataCoSupplyChain.csv` — data dictionary

### Dataset Details

| Attribute | Value |
|-----------|-------|
| Records | ~180,519 |
| Columns | 53 |
| Time Period | 2015–2018 |
| File Size | ~91 MB |
| Encoding | ISO-8859-1 |
| Source | Mendeley Data |
| Citation | Constante, Fabian (2019) |

### Folder Structure

- `raw/` — Original unmodified dataset
- `processed/` — Cleaned and feature-engineered data (generated by notebooks)

### Loading in Python

```python
import pandas as pd
df = pd.read_csv('data/raw/DataCoSupplyChainDataset.csv', encoding='ISO-8859-1')
```

Note the `ISO-8859-1` encoding — the file will fail to load with default UTF-8.
