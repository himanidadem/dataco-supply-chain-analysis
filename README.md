# DataCo Supply Chain Intelligence 📊

End-to-end analysis of supply chain demand, fulfillment, and profitability using the DataCo Global dataset (~180K records).

**Status:** 🚧 Work in progress

## Project Overview

This project analyzes supply chain performance across three key dimensions:

- **Demand Analysis** — Revenue drivers, customer segmentation, seasonality
- **Fulfillment Operations** — Late delivery patterns, root causes
- **Profitability** — Margin analysis by product, region, and shipping mode

## Tech Stack

- **Python** — Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
- **PostgreSQL** — Complex business queries
- **Power BI** — Interactive executive dashboard
- **Git/GitHub** — Version control

- Source: [Mendeley Data](https://data.mendeley.com/datasets/8gx2fvg2k6/5)
- ~180,519 records | 53 columns | 2015–2018
